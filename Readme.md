# EasySpider: Visual Code-Free Web Crawler

一个可视化爬虫，可以使用图形化界面，无代码可视化的设计和执行爬虫任务，支持中文和英文版本。

A visual code-free/no-code web crawler/spider, supporting both Chinese and English.
